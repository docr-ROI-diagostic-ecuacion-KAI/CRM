# CRM Doc ROI Canal Estudiantes

Primera version del CRM Doc ROI para seguimiento de envios al canal estudiante.

Estado actual: repositorio inicializado desde ChatGPT y preparado para recibir la app completa.

Base de datos objetivo: Google Sheet facilitada por Jorge Lucio.

Logo oficial: Logo Doctor ROI facilitado por Jorge Lucio.

Funcion prevista de la version 1:

- Alta de ficha de alumno.
- Nombre del alumno.
- Alias.
- Telefono.
- Buyer Persona.
- Nombre de campana.
- Envio de comunicacion: si o no.
- Fecha de envio.
- Respuesta.
- Respuesta recibida: si o no.
- Navegacion ficha a ficha con flechas.
- Nueva ficha.
- Crecimiento de columnas por campana.
- Conexion con Google Sheets desde Vercel mediante credenciales privadas de servidor.

Nota tecnica: la app completa esta preparada en el ZIP generado en ChatGPT. El conector de GitHub ha permitido escribir archivos de documentacion, pero esta bloqueando archivos tecnicos ejecutables. Por seguridad, las credenciales de Google Sheets no deben subirse al repositorio. Deben configurarse como variables privadas en Vercel.
